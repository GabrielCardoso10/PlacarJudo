Controles Gerais da Luta

P - Inicia/Pausa o cronômetro (normal e Golden Score).

R - Reseta completamente o placar para uma nova luta.

O - Inicia Osaekomi para o Competidor 1 (Branco).

L - Inicia Osaekomi para o Competidor 2 (Azul).

Q - Cancela/Interrompe a contagem de Osaekomi.

Competidor 1 (Faixa Branca)

A / W / S - Adiciona Ippon / Waza-ari / Yuko.

D - Adiciona Shido.

Shift + A / W / S - Remove Ippon / Waza-ari / Yuko.

Z - Remove o último Shido.

Competidor 2 (Faixa Azul)

↓ / ↑ / ← - Adiciona Ippon / Waza-ari / Yuko.

→ - Adiciona Shido.

Shift + ↓ / ↑ / ← - Remove Ippon / Waza-ari / Yuko.

M - Remove o último Shido.
